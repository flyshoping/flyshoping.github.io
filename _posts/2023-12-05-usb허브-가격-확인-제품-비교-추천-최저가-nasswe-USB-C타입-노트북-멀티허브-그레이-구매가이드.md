**USB 허브의 장점**

USB 허브는 하나의 USB 포트에 여러 개의 USB 기기를 연결할 수 있도록 하는 장치입니다. 데스크탑이나 노트북의 USB 포트가 부족할 때 유용하게 사용할 수 있습니다.

USB 허브의 장점은 다음과 같습니다.

* **USB 포트 수 증가**

USB 허브를 사용하면 데스크탑이나 노트북의 USB 포트 수를 늘릴 수 있습니다. 따라서 마우스, 키보드, 외장하드, 프린터 등 다양한 USB 기기를 동시에 연결할 수 있습니다.

* **휴대성**

USB 허브는 일반적으로 작은 크기와 가벼운 무게로 설계되어 있습니다. 따라서 휴대하기가 간편하여 외부에서 사용할 때도 편리합니다.

* **가격**

USB 허브는 비교적 저렴한 가격에 구입할 수 있습니다. 따라서 USB 포트가 부족한 경우 경제적인 방법으로 문제를 해결할 수 있습니다.

USB 허브를 구입할 때는 다음과 같은 사항을 고려해야 합니다.

* **포트 수**

연결할 USB 기기의 수를 고려하여 포트 수를 선택해야 합니다.

* **전원 공급 방식**

연결할 USB 기기의 소비 전력을 고려하여 전원 공급 방식을 선택해야 합니다.
* **지원하는 USB 버전**
